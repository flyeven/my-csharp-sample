页面迁移例子

默认情况下使用 Ajax

使用 Ajax ， 如果 页面加载失败， 将保持当前画面不变， 弹出 提示的错误信息。 而不是像普通电脑的浏览器那样， 如果页面不存在， 弹个空白的 404 错误窗口。 一个不存在的页面

手动禁用 Ajax

访问其他域名的页面， 也 不会使用 Ajax. ( 例如访问 http://www.jquery.com ) 通过 rel="external" data-ajax="false" 标记， 来 禁用 Ajax 的加载. 一般 rel="external" 使用在 同一个域名下， 其他目录下的页面。 data-ajax="false" 用于同域名， 同级目录下的页面。 一个不存在的页面

返回上一页

如果 使用了 data-rel="back"， 那么 将返回上一页， 并忽略 href 中所填写的内容. 返回上一页 ( 忽略 href )

弹出对话框

作为一个 对话框 画面， 不希望被加入到 浏览的历史列表中 也就是在 浏览器的 前进/后退的时候， 不希望看到这个 对话框的画面。 处理办法： 增加 data-rel="dialog" 的属性。 对话框

使用/禁用 Ajax 效果对比
