Inline buttons

默认情况下， 一个按钮是独占一行的， 效果如下：

Button

通过 data-inline="true" 来使其不独占满一行显示

Button

这样， 多个按钮可以并在一行显示 


<a href="index.html" data-role="button" data-inline="true">Cancel</a>
<a href="index.html" data-role="button" data-inline="true" data-theme="b">Save</a>

结果如下：

Cancel Save

加了 data-mini="true" 的 mini 版本显示效果如下：

Cancel Save
